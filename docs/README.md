# Project Ed Internal Docs

Internal source-of-truth documentation for Project Ed lives here.

## Rule

If it is internal and reusable, it belongs in Markdown in this repo.

Google Docs is for client-facing deliverables.
ClickUp is for tasks, checklists, and commitments.
ChatGPT project conversations are working sessions, not durable documentation.

## Triage buckets

When auditing old notes and docs, sort each item into one of these buckets:

1. Source-of-truth internal docs -> move here as Markdown.
2. Client-facing deliverables -> keep in Google Docs.
3. Task tracking -> convert to ClickUp task/checklist.
4. Old/no longer useful -> archive/dungeon.
